Unpivot Avaya CM agent list

A script for transposing the CSV output of Avaya CM's agent list report for a more practical tabular layout suitable for pivot tables, etc

Download this project as a .zip file Download this project as a tar.gz file

The problem

The format of the agent list report (snapshot w/ skill assignments) from Avaya CM/GEDI is unwieldy, this is an F# script that tries to fix that by transforming the CSV output of the report.

Requirements

  • Avaya Site Administrator installed on your system (tested with ver. 6.0)
  • Credentials for logging into GEDI and the ability to run and export the 'agent list' report
  • ALTERNATIVE: access to a CSV file representation of the agent list report exported by someone with the above privileges following the steps outlined in 'Getting the data'
  • ALTERNATIVE: If you just want to see how the script works without your own data, a sample input data file is provided in the repository, as outlined in 'Getting the data'
  • OPTIONAL: Visual Studio 2013 Community Edition (FREE) or higher if you want to build from source or run the script in an Interactive Window

Getting the data

OPTION A: Retrieving your data from Avaya CM/GEDI

We'll export a CSV representation of the Agent List report from Avaya Site Administrator:

  • Launch Avaya Site Administrator
  • Login to GEDI
  • Run the agent list snapshot report by typing in 'li ag' in the report execution command box
  • Once the report has finished loading, click on File > Export
  • In the dialog box, select a memorable name and location for the exported report. Make sure you are saving as .csv.
  • Ensure the 'Export column titles on first row?' check box is checked and click OK
  • You should now have a CSV file with the raw report format, next run the script on the file.
